# Novel IF Reader

`plan.md`의 MVP 범위를 정적 브라우저 앱으로 구현한 프로토타입입니다. 원문 텍스트를 입력하면 문단과 장면을 나누고, 인물·장소·사건·상태·공간 그래프를 추출한 뒤 독서 위치 기준으로 스포일러를 차단해 보여줍니다.

## 실행

브라우저에서 `index.html`을 열면 됩니다. 별도 설치나 서버가 필요 없습니다.

## 포함 기능

- 원문 입력 및 샘플 로드
- 문단 기반 세그먼트와 장면 생성
- 규칙 기반 인물, 장소, 사건 추출
- 독서 위치 슬라이더와 스포일러 차단
- 공간 그래프, 사건 타임라인, 인물 상태 카드
- 인물·장소·사건 수동 수정
- JSON, CSV, Markdown 내보내기

## 제약

현재 분석기는 LLM이 아니라 로컬 규칙 기반 추출기입니다. 결과는 근거 문장과 확실성 표시를 포함하지만, 한국어 문학 텍스트의 모호한 해석은 사용자가 Review 탭에서 수정하는 전제를 둡니다.
