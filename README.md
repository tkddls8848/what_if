# Novel IF Reader

`plan.md`의 MVP 범위를 정적 브라우저 앱으로 구현한 프로토타입입니다. 이상 「날개」 원문은 `texts/wings.txt`에 분리했고, 앱 시작 시 `fetch()`로 불러와 샘플 텍스트로 사용합니다.

## 실행

텍스트 파일을 브라우저에서 불러와야 하므로 로컬 서버로 실행하세요.

```powershell
cd D:\novel_if
python -m http.server 8000
```

그다음 브라우저에서 아래 주소를 엽니다.

```text
http://localhost:8000/
```

`node app.js`는 실행 방식이 아닙니다. `app.js`는 브라우저 DOM에서 동작하는 프론트엔드 스크립트입니다.

## 포함 기능

- `texts/wings.txt`에서 이상 「날개」 원문 로드
- 원문 입력 및 샘플 다시 로드
- 문단 기반 세그먼트와 장면 생성
- 규칙 기반 인물, 장소, 사건 추출
- 독서 위치 슬라이더와 스포일러 차단
- 현재 타임라인에 서술된 장소만 표시하는 공간 그래프
- 사건 타임라인, 인물 상태 카드
- 인물·장소·사건 수동 수정
- JSON, CSV, Markdown 내보내기

## 제약

현재 분석기는 LLM이 아니라 로컬 규칙 기반 추출기입니다. 결과는 근거 문장과 확실성 표시를 포함하지만, 문학 텍스트의 모호한 해석은 사용자가 Review 탭에서 수정하는 전제를 둡니다.
